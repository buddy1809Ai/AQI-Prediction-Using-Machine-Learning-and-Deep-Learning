# Contributing

1. Fork → `git checkout -b feature/my-improvement`
2. Make changes, ensure `pytest tests/ -v` passes
3. Commit with clear messages, open a Pull Request

## Standards
- PEP 8 · Python 3.11 · Docstrings on all public functions
- No magic numbers — use `app/config/settings.py`
- No absolute paths

## Areas
- More cities · CatBoost/LightGBM models · SHAP visualizations · Real-time data
